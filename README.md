# 🤖 AI Data Analysis Assistant

Transform raw data into actionable insights using AI. Upload your CSV or Excel dataset, ask questions in plain English, and instantly receive SQL queries, interactive visualizations, KPI dashboards, and AI-powered business insights.

## 📌 Features

### 🔍 Natural Language to SQL
- Ask questions in plain English.
- Automatically generates SQL queries using Gemini AI.
- Executes queries against uploaded datasets.

### 📊 Interactive Data Visualization
- Dynamic chart generation based on user requirements.
- Supports:
  - Bar Charts
  - Pie Charts
  - Line Charts
  - Scatter Plots

### 📈 KPI Dashboard
- Total Rows
- Total Columns
- Dataset Size
- Query Result Metrics

### 🧠 AI-Powered Insights
- Generates business insights from query results.
- Provides actionable recommendations and trends.

### 📄 Professional PDF Reports
- Export analysis results into professional reports.
- Includes:
  - Query Results
  - Visualizations
  - KPI Metrics
  - AI Insights

### 📱 Responsive Design
- Desktop Support
- Tablet Support
- Mobile-Friendly Interface

### ☁️ Cloud Deployment
- Hosted on Streamlit Cloud.
- Secure API key management using Streamlit Secrets.

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/Dinaz-12/AI-Data-Analysis-Assistant.git

cd AI-Data-Analysis-Assistant
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

Windows

```bash
venv\Scripts\activate
```

Mac/Linux

```bash
source venv/bin/activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Configure Environment Variables

Create a `.env` file:

```env
GEMINI_API_KEY=YOUR_API_KEY
```

### Run Application

```bash
python -m streamlit run streamlit_app.py
```
